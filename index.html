The Road Layer Detection project is designed to analyze road construction progress by comparing two images: one before and one after the base and subgrade phase. The app detects road layers by calculating their thickness and volume using point cloud data. It processes the data, subtracts point cloud values to determine the thickness, and computes the volume, with the point cloud files being deleted once the process is complete.
