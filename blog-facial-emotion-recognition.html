Facial Emotion Recognition with CNN

Admin

Computer Vision, Deep Learning

10-June-2024

Project Overview

I built a real-time Facial Emotion Recognition (FER) system using a Convolutional Neural Network (CNN). The model classifies facial expressions into emotions such as happy, sad, angry, surprise, fear, disgust, and neutral from images or a live camera feed. The application runs efficiently and can be integrated into user feedback systems, education tools, or smart kiosks.

Dataset
  • Preprocessed grayscale face images resized to a fixed input size (e.g., 48x48).
  • Balanced classes via augmentation (horizontal flip, slight rotation, brightness/contrast jitter).
  • Train/validation/test split with stratification.
CNN Architecture
  • Stacked Conv2D blocks with BatchNorm and ReLU activations.
  • MaxPooling for spatial downsampling and Dropout for regularization.
  • Fully connected layers with softmax output for multi-class emotion prediction.
Training & Optimization
  • Optimizer: Adam with step or cosine learning-rate scheduling.
  • Loss: Categorical Cross-Entropy; metrics: accuracy, F1 per class.
  • Early stopping and model checkpointing to avoid overfitting.
Results
  • Strong accuracy on validation set with robust performance across major classes.
  • Confusion-matrix analysis used to reduce misclassification between similar expressions (e.g., fear vs surprise).
  • Real-time inference on CPU/GPU with OpenCV face detection and stable FPS.
Demo & Usage
  • Webcam stream: detect face, preprocess, predict emotion, and overlay label.
  • Image mode: upload a photo and get the predicted label with confidence.
  • Modular design to plug in improved detectors (e.g., RetinaFace) or advanced backbones later.
Future Work
  • Upgrade to deeper CNNs or Transformer-based vision models (ViT) for higher accuracy.
  • Temporal smoothing over video frames for more stable predictions.
  • Bias and fairness evaluation across diverse demographics.
