My Eyes – Assistive Vision App

Admin

Mobile App, AI

23-Jan-2025

My Eyes is an assistive mobile application designed to help blind and visually impaired users understand their surroundings. Built with Flutter, the app opens the camera and uses an on-device YOLO model to detect objects in real time. Detected objects are spoken out loud, enabling users to hear what is in front of them with minimal latency.

Key Features
  • Real-time object detection using a YOLO model integrated in the app
  • Text-to-Speech (TTS) narration that speaks detected objects clearly
  • Optimized camera pipeline for smoother detection on mobile hardware
  • Clean and accessible UI built with Flutter
  • On-device inference for privacy and offline usage
Tech Stack
  • Flutter (Dart) for cross‑platform mobile UI
  • YOLO object detection model (mobile‑optimized)
  • Platform channels / plugins for camera stream and TTS
Outcome

The project was executed as a Final Year Project (FYP) for our client. It received recognition and an HEC stipend of PKR 80,000, acknowledging its social impact and technical quality.
