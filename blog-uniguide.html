UniGuide – Personalized University & Scholarship Recommendations

Admin

Recommender Systems, AI

8-April-2025

Project Overview

UniGuide is a personalized recommendation system that suggests universities and relevant scholarships based on a user's profile and preferences. It considers program interests, budget constraints, location, academic scores, language tests, and deadlines to produce ranked recommendations.

Key Features
  • Collects user preferences (interests, budget, GPA/grades, etc).
  • Ranks universities by fit score using multi‑criteria scoring and filters.
  • Recommends scholarships matched to the user's profile and deadlines.
  • Explains why each recommendation is a good fit (transparent scoring).
Data & Method
  • Aggregated program catalogs and scholarship datasets with standardized fields.
  • Hybrid approach: rules/filters for hard constraints and a learned model for ranking (e.g., gradient boosting or neural scoring).
Tech Stack
  • Backend: Python (FastAPI) with a modular recommendation pipeline.
  • Ranking: scikit‑learn for fit scoring.
  • Database: PostgreSQL (normalized program and scholarship tables).
  • Frontend: Responsive UI.
How It Works
  1. Collect user inputs and apply filters (eligibility, budget, location, deadlines).
  2. Compute a fit score per program using weighted criteria (scores, cost, ranking).
  3. Match scholarships using eligibility rules and rank by relevance and deadline.
  4. Present top programs and scholarships with explanations and quick apply links.
Future Enhancements
  • Integrate live APIs for rankings, tuition, and scholarship feeds.
  • Personalized weights learned from user feedback.
  • Chat assistant to refine preferences and answer queries.
